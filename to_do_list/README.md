ToDo List Application
Overview
This is a straightforward ToDo list application developed using the Flask web framework. The application offers users a simple interface to manage their tasks, allowing them to add, view, and delete items from their to-do list.

Video Demo: https://youtu.be/QlTQbt5sP3I
Features
The application boasts the following key features:

Add New Tasks: Users can easily add new tasks to their to-do list, ensuring a quick and intuitive input process.

View Task List: The application provides a clear and organized display of all tasks in the to-do list, enhancing user experience and task management.

Delete Tasks: Users have the flexibility to remove tasks from the list, facilitating efficient task organization.

Prerequisites
Before diving into the application, make sure your system meets the following prerequisites:

Python: Ensure that you have Python installed on your system, with version 3.x recommended for optimal compatibility.

Flask: Install the Flask web framework to support the functionality of the application.

Getting Started
Follow these steps to get started with the ToDo list application:

Clone the Repository:

bash
Copy code
git clone https://github.com/trevdav1972
This command will clone the application repository to your local machine.

Install Dependencies:

Navigate to the project directory and install the necessary dependencies, including Flask.

bash
Copy code
cd your_project_directory
pip install -r requirements.txt
Run the Application:

Start the Flask development server by running the following command in your terminal:

bash
Copy code
flask run
This will initiate the application, making it accessible through your web browser.

Interact with the ToDo List:

Open the application in your web browser, and you can now add, view, and delete tasks from your to-do list. Enter tasks, manage your schedule, and streamline your daily activities effortlessly.

Additional Notes
Feel free to explore and customize the application to suit your preferences. The simplicity of the Flask framework allows for easy modifications and enhancements. Enjoy the efficient task management offered by this ToDo list application!
